"Hello World!"
Kad programmētāji pirmo reizi uzsāk mācīties jaunu programmēšanas valodu, tad viena no pirmajām programmām ko viņi izveido ir programma kas izprintē tekstu "Hello World!".
No kā sastāv šī programma?
#Include <iostream> un #include "pch.h"
Šie elementi ievieto C++ standarta bibliotēku, lai jūs varētu izmantot jau iepriekš sagatavotus elementus. Specifiski Iostream ir ieejas un izejas (InputOutput) bibliotēka, kas var gan printēt tekstu, gan saņemt to no lietotāja
int main()
Šo elementu sauc par funkciju, specifiskāk, galveno funkciju jo tās nosaukums ir main. Int main() nosaka C/C++ programmas koda ievadi vai sākumpunktu. Šeit int ir atslēgvārds, kas stāsta par vesela skaitļa datu tipu. Tātad, kad kāds palaiž failu savā lokālajā datorā, operētājsistēma izpilda galveno () funkciju.
std::cout << "Hello World!";
std::cout ir izteiciens, kura funkcija ir informācijas printēšana uz ekrāna. << ir praktiski tas pats, kas = zīme. Ievietotais teksts pēdiņās būs teksts, kas tiks izprintēts uz ekrāna, kad programma tiks atvērta.
return 0;
return 0; ir kā tests, lai saprastu vai programma ir izgājusi cauri visam kodam. Ja tā ir tad konsolē būs parādīts:
... exited with code 0.
